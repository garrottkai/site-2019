About this site

The globe on this page displays an automatically updated, 1x1-degree grid representing the relative amount of vegetation on the Earth's surface. This data is collected by NASA's MODIS sensor suite on the TERRA and AQUA earth-observing satellites. The dataset is updated on NASA's servers approximately every 16 days, and is automatically retrieved and processed by a script for display.

The model was built in Three.js, a 3D graphics utility for the browser, which provides a convenient API for creating hardware-accelerated animations with WebGL.
